# nyan-cat cookbook
Than nyan-cat cookbook configures a node to use Andrea Campi's
nyan-cat-chef-formatter by default when running chef-client.

# Requirements
A sense of humor

# Usage
Include "recipe[nyan-cat]" in a node's run_list
